# Security Policy

## Supported Versions
Only the latest version of the `zkv-control-center` (`main` branch) receives security updates.

## Reporting a Vulnerability
If you discover a security vulnerability regarding OPSEC, keys management, or port exposure within this tool, please **do not** create a public issue. 
Instead, reach out directly via [Tu Usuario de Telegram/Discord o Email]. I take security seriously and will issue a patch within 24 hours.
